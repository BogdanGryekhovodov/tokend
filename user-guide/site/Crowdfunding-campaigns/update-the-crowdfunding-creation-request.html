    Update the crowdfunding creation request

    Note: you can update any data of your request at any time if it's pending, but you cannot â€“ if it's permanently rejected.

    1. In the navigation bar, go to Requests
    2. Go to the Fund creation tab
    3. Optionally, in the Asset list in the upper-left corner of the page, select the asset
    4. Select the request you want to update
    5. Click on the icon on the right
    6. Click Update
    7. You will be redirected to the Create sell page, where you can update any data of this request (so you can follow the same crowdfunding campaign creation guide)
